According to XKCD, if you start from any random article, 
click on the first link in the article text not in parentheses or italics, and repeat,
you will eventually reach the Philosophy page. 

This web crawler, built using the Scrapy framework for Python, was designed to automate this process.

To run the spider, navigate to the top-level directory containing scrapy.cfg in the command prompt 
and run scrapy crawl wiki.
